Welcome to the Theory of Computation web application!

This application provides tools for working with various concepts in theory of computation, including:

  • Deterministic Finite Automata (DFA)
  • Non-deterministic Finite Automata (NFA)
  • Context-Free Grammars (CFG)
  • Pushdown Automata (PDA)
  • Regular Expressions (Regex)
